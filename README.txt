## Funcionalidades

1. **Adicionar Registro**: Permite adicionar um novo registro de clique, especificando o timestamp, a posição X e a posição Y do clique.

2. **Exibir Registros**: Exibe todos os registros de cliques armazenados no arquivo CSV.

3. **Atualizar Registro**: Permite atualizar um registro existente de clique, especificando o índice do registro a ser atualizado, bem como o novo timestamp, posição X e posição Y.

4. **Excluir Registro**: Permite excluir um registro existente de clique, especificando o índice do registro a ser excluído.

5. **Sair**: Encerra o programa e salva todas as alterações feitas nos registros de cliques.

## Como Usar

1. Execute o arquivo `meu_programa.py` em um ambiente Python.

2. O menu será exibido, apresentando as opções disponíveis.

3. Escolha a opção desejada digitando o número correspondente e pressione Enter.

4. Siga as instruções apresentadas na tela para realizar a operação desejada.

5. Ao selecionar a opção "Sair", todas as alterações feitas nos registros serão salvas no arquivo `dados_navegacao.csv`.

## Requisitos

- Python 3.x instalado no sistema.

- O arquivo `dados_navegacao.csv` deve estar presente no mesmo diretório que o arquivo `meu_programa.py`.

## Observações

- Este projeto foi desenvolvido com base na entrega de um Chatbot, adaptando o conceito de mapeamento de cliques para um ambiente Python.

- Certifique-se de seguir as instruções apresentadas na tela para cada opção do menu.
